Addressing Standard in the UAE

 

This is the standard format to be used when writing an address (senders & recipients address).

 

1- For sending letters, registered mail and parcels to a personal P.O Box

To,

Title & Name

P.O Box

Emirate

UAE

 

2- For sending letters, registered mail and parcels to a corporate P.O Box

To,

Title & Name

Company Name

Department Name

P.O Box

Emirate

UAE

 

3- For sending Express Mail:

To,

Title & Name

Company Name

Department Name

P.O Box

Physical Address (Building Name, Street Name, Flat number)

Emirate

UAE

Telephone number
